Shell 初级笔记

教程来源：
shell tutorial

shell 可视为 linux 命令的脚本文件，对于一些复杂的处理当然还是 选择 c++ / c 或者其他语言会更方便。

读完之后给自己布置个作业1（简单版）：

使用 bash 写一个linux 的 IO 测试脚本，并附上说明。

记录写这个作业遇到的一些问题：

  1. [ ] 左右需要空格隔开
  2. 为了防止字符串为空的情况 最好使用 加上不影响结果的占位符防止意外
  3. cd 这个指令 并不会在命令行切换路径
