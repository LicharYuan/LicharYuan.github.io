关于 diffusion， 推荐这位大佬的 Diffusion博客

回顾一下diffusion做的事情：

  • forward: $x_0$ (原图) -> Guass_noise(0, $i_1$) -> $x_1$ -> Guass_noise(0,$i_2$) … -> $x_N$
  • backward: $x_N$ -> condition_bayes + reparameter -> $x_{N-1}$ -> … -> $x_0$

从 0 到 N 的 forward 过程，可以使用 $q(x_N | x_{N-1})$ 来表示；从 N 到 0 的 backward 过程用 $p_\theta(x_{N-1} | x_{N})$ 来表示；前向的参数是确定的，就像无法预测未来，我们也无法估计 $q(x_{N-1} | x_{N})$；但是高斯参数
